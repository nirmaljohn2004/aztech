Project: LED Screen UAE – Professional Website Redesign
Platform: Desktop & Mobile Website
Goal: Redesign the existing website into a clean, modern, professional, and premium-looking website that clearly communicates the company’s standard and core business: selling LED screens in UAE.

Design Objective
Create a high‑end corporate website similar in quality, layout clarity, and professionalism to https://www.vrdgroups.com/.
When the website opens, users must instantly understand:

The company sells LED screens
The company operates in UAE
The company is professional, reliable, and established
Remove all unnecessary content and focus only on information that builds clarity, trust, and conversion.

Visual Style & Brand Feel
Modern, premium, corporate
Minimal and content‑focused
High contrast, clean spacing
Confident and trustworthy (not flashy)
Color Palette
Primary background: Dark charcoal / near‑black
Primary text: White
Secondary text: Light gray
Accent color: Red or Blue (LED glow style)
Use accent color only for CTAs, highlights, and focus elements
Typography
Headings: Poppins or Inter (Bold / Semi‑Bold)
Body text: Inter or Roboto (Regular / Medium)
Clear hierarchy, large headings, readable body text
Website Structure
Header / Navigation
Logo on the left
Navigation: Home, About, Products, Projects, Contact
Primary CTA button: Get a Quote
Clean, fixed header style
Home Page Layout
